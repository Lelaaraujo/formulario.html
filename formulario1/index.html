  2. Perguntas e Respostas
    • Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?
      • Cumpre a função de fornecer ao usuário uma dica que descreve o valor esperado em um dado campo de entrada.

    • Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
      • Cumpre a função de definir se o componente está checado.

    • Na tag < input > para que serve o atributo "value"?
      • Cumpre a função de definir o valor enviado por este item.

    • Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
      • Você deve usar o atributo value para definir o valor a ser enviado por este item. Use o atributo checked para indicar se este item deve estar selecionado por padrão.

    • Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?
      • Label= definir um rótulo para elementos do formulário. O atributo for faz um tipo de link entre a label e o input . Logo o for tem que apontar para o ID do input .

    • Na tag < option > para que serve o atributo "selected"?
      • Permite especificar que um controle de formulário deve ter foco de entrada quando a página é carregada, a não ser que o usuário o substitua, por exemplo digitando em um controle diferente.

    • Nas tags de formulário para que serve o atributo "required"?
      • O atributo required indica que esse campo é obrigatório, e portanto, a opção selecionada não deve ter valor vazio.

    • Para que serve a tag < textarea >?
      • Representa um controle de edição para uma caixa de texto, útil quando você quer permitir ao usuário informar um texto extenso em formato livre, como um comentário ou formulário de retorno.

    • Na tag < form > para que serve o atributo "action"?
      • Define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados

    • Na tag < form > para que serve o atributo "method"?
      • Define qual o método HTTP para enviar os dados

    • Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >?
      • Deve-se usar o valor POST

    • Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?
      • O atributo ACTION é quem diz para onde os dados de um formulário deve ser enviado.

    • O que as tags < fieldset > e < legend > produzem?
      • a tag < fieldset > é usada para agrupar elementos, assim como a tag < label >, dentro de um formulário web. Já a tag < legend > define uma legenda para o elemento < fieldset >.
